## How to Use

1. **Start with strategy/** — understand the pillars, algorithm approach, and profile setup
2. **Follow calendar/** — day-by-day posting plan with topics and times
3. **Film from scripts/** — each script has full shot-by-shot instructions, text overlays, captions
4. **Execute engagement/** — daily 30-min routine for growth
5. **Review analytics/** — weekly check-ins, monthly reports
