# Analytics & KPI Framework

## Core Metrics Dashboard

| Metric | Formula | Week 1-4 Target | Month 2 Target | Month 3 Target | Action if Below |
|--------|---------|----------------|---------------|----------------|----------------|
| Follower Growth | New followers / week | +100-200/week | +500/week | +1,500/week | Post more, improve hooks |
| Avg Views/Post | Total views / posts | 500-1,000 | 2,000-5,000 | 5,000-15,000 | Better hooks, trending sounds |
| Completion Rate | Avg watch time / video length | >40% | >50% | >55% | Shorten videos, tighten pacing |
| Share Rate | Shares / Views | >0.5% | >1% | >1.5% | More wow factor, useful content |
| Save Rate | Saves / Views | >1% | >2% | >3% | More tutorials, tip content |
| Comment Rate | Comments / Views | >2% | >3% | >4% | Ask more questions, controversy |
| Profile Visit Rate | Profile visits / Views | >1% | >2% | >3% | Stronger CTAs, curiosity hooks |
| Link Clicks | Click-throughs from bio link | 10+/day | 50+/day | 200+/day | Better CTA in bio + videos |

### Step 2: Analyze Patterns
- Which pillar had the highest avg views?
- Which post format had the best completion rate?
- What time of day performed best?
- Did any series episode break out? Which one?
- Were trend rides outperforming evergreen content?

### Step 3: Adjust Next Week
- Increase posts in the top-performing pillar
- Reduce or rethink the worst-performing pillar
- Double down on any breakout format
- Swap posting times if data suggests better windows

## Attribution: TikTok → App Installs

### Tracking Setup
- Use UTM-tagged links in bio: `yourapp.com/tiktok?utm_source=tiktok&utm_medium=organic`
- Track link clicks in TikTok Analytics (Creator Tools → Content → Link Clicks)
- Cross-reference with App Store Connect / Google Play Console install spikes
- Look for install spikes that correlate with high-performing TikTok posts
