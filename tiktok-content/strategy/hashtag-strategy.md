## Hashtag Usage Rules

1. **3-5 hashtags per post** (TikTok algorithm penalizes more than 5)
2. **Formula per post:** 1 Brand + 1-2 Core Niche + 1 from another cluster + 0-1 Trending
3. **Never use the exact same set twice in a row** — rotate systematically
4. **Place hashtags in caption**, not comments (TikTok processes caption hashtags for distribution)
5. **No generic mega-hashtags** (#fyp, #foryou, #viral — these are too broad to help)
